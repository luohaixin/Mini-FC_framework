# 网络视频下载工具

一个功能完善的网络视频下载工具，支持多种视频网站，提供高清下载和在线预览功能。

## 功能特性

1. **直观的用户界面**
   - 简洁现代的 UI 设计
   - 响应式布局，支持移动端
   - 清晰的操作流程

2. **视频链接解析**
   - 支持 100+ 视频网站
   - 自动识别视频资源信息
   - 提取视频标题、时长、上传者等信息

3. **视频格式筛选**
   - 仅展示视频相关格式
   - 支持多种清晰度选择 (720P, 1080P, 4K 等)
   - 显示文件大小和编码信息

4. **在线预览功能**
   - 下载前预览视频内容
   - 确认视频完整性和正确性
   - 支持多种视频格式播放

5. **下载管理**
   - 实时显示下载进度
   - 支持自定义文件名
   - 后台异步下载
   - 下载完成后自动保存

6. **支持的视频网站**
   - YouTube
   - Bilibili (哔哩哔哩)
   - 抖音
   - 快手
   - 优酷
   - 爱奇艺
   - 腾讯视频
   - 微博视频
   - 以及 100+ 其他网站

## 技术栈

- **后端**: Python + Flask
- **视频解析**: yt-dlp
- **前端**: HTML5 + CSS3 + JavaScript
- **UI 框架**: Font Awesome Icons

## 安装说明

### 1. 克隆或下载项目

```bash
cd 视频下载
```

### 2. 安装依赖

```bash
pip3 install -r requirements.txt
```

### 3. 运行应用

```bash
python3 app.py
```

### 4. 访问应用

打开浏览器访问: http://localhost:5000

## 使用说明

1. **输入视频链接**
   - 在输入框中粘贴视频链接
   - 支持 YouTube、Bilibili、抖音等多种网站

2. **解析视频**
   - 点击"解析链接"按钮
   - 等待系统提取视频信息

3. **预览视频**
   - 解析完成后会自动加载视频预览
   - 可以播放视频确认内容

4. **选择格式**
   - 从可用格式中选择所需的清晰度
   - 显示文件大小和格式信息

5. **开始下载**
   - 可选：自定义文件名
   - 点击"开始下载"按钮
   - 查看下载进度
   - 下载完成后自动保存到 downloads 目录

## 注意事项

1. **网络要求**
   - 需要能够访问目标视频网站
   - 部分网站可能需要代理

2. **存储空间**
   - 确保有足够的磁盘空间存储下载的视频
   - 默认下载目录: `downloads/`

3. **合法性**
   - 请遵守相关网站的使用条款
   - 仅下载您有权下载的内容
   - 尊重版权和知识产权

## 故障排除

### 无法解析链接
- 检查链接是否正确
- 确认能够访问该视频网站
- 部分网站可能需要更新 yt-dlp

### 下载失败
- 检查网络连接
- 确认磁盘空间充足
- 查看错误提示信息

### 预览无法播放
- 部分视频格式可能不被浏览器支持
- 尝试下载后再播放
- 检查网络连接
